# Kana Trainer

일본어 IME 입력 없이 히라가나와 가타카나를 연습하는 작은 학습 앱입니다.
터미널풍 GUI와 CLI를 모두 지원하며, 문자 읽기에서 단어 뜻 맞히기까지 단계적으로 연습할 수 있습니다.

- 현재 버전: `0.3.0`.
- 지원 환경: Windows와 Python 실행 환경.
- 기본 데이터: 저장소의 `일본어.md`와 앱 내장 가나·단어 데이터.

## 빠른 실행

Windows에서는 배치 파일로 GUI를 바로 실행할 수 있습니다.

```powershell
.\run-kana-trainer.bat
```

Python 명령으로 실행하려면 아래 명령을 사용합니다.

```powershell
python -m kana_trainer --gui
```

CLI 모드로 실행하려면 옵션 없이 실행합니다.

```powershell
python -m kana_trainer
```

입력 없이 간단히 동작을 확인하려면 데모 모드를 사용할 수 있습니다.

```powershell
python -m kana_trainer --demo
```

## 주요 기능

- 히라가나와 가타카나 로마자 입력 게임.
- 로마자 보고 히라가나 4지선다 선택.
- 히라가나와 가타카나 매칭.
- 단어를 보고 한국어 뜻 맞히기.
- 조사 뜻 맞히기.
- 헷갈리는 문자 2지선다 선택.
- 틀린 문자 오답 복습.
- 학습 기록과 정답률 요약.
- `일본어.md` 기반 참고 자료와 원문 보기.

## 가나 레벨

히라가나와 가타카나 로마자 입력 게임은 4단계로 진행됩니다.

| 레벨 | 내용 | 설명 |
|---|---|---|
| Lv.1 | 기본 | 기본 오십음 문자입니다. |
| Lv.2 | 탁음·반탁음 | `が`, `ぱ`처럼 점이나 동그라미가 붙은 문자입니다. |
| Lv.3 | 작은 글자 조합 | `きゃ`, `ショ`처럼 작은 글자가 결합된 문자입니다. |
| Lv.4 | 단어 뜻 맞히기 | 단어를 보고 한국어 뜻을 고르는 4지선다입니다. |

Lv.1은 항상 열려 있습니다.
이전 레벨에서 정답률 80% 이상을 기록하면 다음 레벨이 열립니다.

## 게임 방식

히라가나와 가타카나 로마자 입력 게임은 목숨 3개로 시작합니다.

- 정답을 맞히면 다음 문제로 넘어갑니다.
- 오답도 한 번 나온 문제로 소비됩니다.
- 목숨이 0이 되면 게임이 종료됩니다.
- 문제를 끝까지 통과하면 게임 승리입니다.

Lv.4 단어 뜻 맞히기는 각 스크립트별 100개 이상의 단어 풀을 사용합니다.
현재 단어 풀은 히라가나 109개, 가타카나 109개입니다.

## 출제 수

| 모드 | 기본 문항 수 | 중복 출제 |
|---|---:|---|
| 가나 입력 게임 | 20문항 | 없음 |
| 로마자 보고 히라가나 선택 | 20문항 | 없음 |
| 히라가나-가타카나 매칭 | 20문항 | 없음 |
| 단어 뜻 맞히기 | 20문항 | 없음 |
| 조사 뜻 맞히기 | 10문항 | 없음 |
| 헷갈리는 문자 선택 | 10문항 | 없음 |

단어 뜻 맞히기는 사용 가능한 문제 풀이 요청 문항 수보다 적으면 풀 크기까지만 출제합니다.

## 저장되는 데이터

오답, 학습 기록, GUI 설정은 사용자 홈의 `.kana-trainer` 폴더에 저장됩니다.

| 데이터 | 기본 경로 | 환경변수 |
|---|---|---|
| 오답 기록 | `.kana-trainer/wrong-answers.json` | `KANA_TRAINER_WRONG_PATH` |
| 학습 기록 | `.kana-trainer/study-history.json` | `KANA_TRAINER_HISTORY_PATH` |
| GUI 설정 | `.kana-trainer/settings.json` | 없음 |

## GUI 참고

GUI에서는 글자 크기와 창 상태를 저장합니다.

- `A-`: 글자 크기 줄이기.
- `A+`: 글자 크기 키우기.
- `Ctrl+마우스 휠`: 글자 크기 조절.

표시 폰트는 `나눔고딕`, `NanumGothic`, `Nanum Gothic`, `Yu Gothic UI`, `Meiryo`, `Yu Gothic`, `MS Gothic` 순서로 사용할 수 있는 폰트를 자동 선택합니다.

## CLI 참고

CLI 대화형 메뉴에서는 터미널 글자 크기 변경 시 입력 줄과 기존 출력이 겹치는 현상을 줄이기 위해 화면을 자동으로 정리합니다.
이 동작을 끄려면 아래 환경변수를 설정합니다.

```powershell
$env:KANA_TRAINER_NO_CLEAR = "1"
python -m kana_trainer
```

## Windows exe 빌드

PyInstaller로 GUI 실행 파일을 만들 수 있습니다.

```powershell
python -m pip install -r requirements-build.txt
python -m PyInstaller kana-trainer.spec
```

빌드 결과는 `dist/KanaTrainer.exe`에 생성됩니다.
`일본어.md`는 실행 파일 내부 리소스로 포함되며, 오답·학습 기록·GUI 설정 JSON은 기존처럼 사용자 홈의 `.kana-trainer` 아래에 생성됩니다.
