히라가나

# 1) 기본 5모음

- **あ い う え お** → **아 이 우 에 오**

# 2) 행(열) 리듬으로 암기 (각 5개 세트)

- **か き く け こ** → **카 키 쿠 케 코**
    
    **が ぎ ぐ げ ご** → **가 기 구 게 고** (탁점)
    
- **さ し す せ そ** → **사 시 스 세 소**
    
    **ざ じ ず ぜ ぞ** → **자 지 즈 제 조** (탁점, *し+゛=じ(지)*)
    
- **た ち つ て と** → **타 치 츠 테 토**
    
    **だ ぢ づ で ど** → **다 지/디 즈/두 데 도** (탁점, *ぢ/づ는 실제로 ‘지/즈’처럼 들림)
    
- **な に ぬ ね の** → **나 니 누 네 노**
- **は ひ ふ へ ほ** → **하 히 후 헤 호**
    
    **ば び ぶ べ ぼ** → **바 비 부 베 보** (탁점)
    
    **ぱ ぴ ぷ ぺ ぽ** → **파 피 푸 페 포** (반탁점)
    
- **ま み む め も** → **마 미 무 메 모**
- **や ゆ よ** → **야 유 요** (이·에 없음)
- **ら り る れ ろ** → **라 리 루 레 로**
- **わ を** → **와 오(조사에서 ‘오’로 읽음)**
- **ん** → **응/ㄴ**(뒤 소리에 따라 ‘응’ 또는 ‘n’ 느낌)

# 3) 작은 글자(ゃゅょ) 결합 — 한 소리로 붙여 읽기

- **きゃ きゅ きょ** → **캬 큐 쿄**
- **しゃ しゅ しょ** → **샤 슈 쇼**
- **ちゃ ちゅ ちょ** → **챠 츄 쵸**
- **にゃ にゅ にょ** → **냐 뉴 뇨**
- **ひゃ ひゅ ひょ** → **햐 휴 효**
- **みゃ みゅ みょ** → **먀 뮤 묘**
- **りゃ りゅ りょ** → **랴 류 료**
- **ぎゃ ぎゅ ぎょ** → **갸 규 교**
- **じゃ じゅ じょ** → **자 주 죠**
- **びゃ びゅ びょ** → **뱌 뷰 뵤**
- **ぴゃ ぴゅ ぴょ** → **퍄 퓨 표**
    
    *(큰 や/ゆ/よ를 쓰면 분리 발음: きよ=키요 vs きょ=쿄)*
    

# 4) 작은 っ(促音, sokuon) — 뒤 자음 **겹치기**

- **がっこう** → **가(ㄲ)꼬-** *(gakkō, ‘학교’)*
- **きって** → **킷테** *(kitte, ‘우표’)*
- **ちょっと** → **춋토/춋또** *(chotto, ‘조금’)*
- **べっさつ** → **벳사츠** *(bessatsu, ‘별책’)*
    
    → 느낌: **자음 한 번 꾹 멈추고** 다음 자음
    

# 5) 탁점(゛)·반탁점(゜) 빠르게

- **K→G**: か→**が**(카→가)
- **S→Z**: さ→**ざ**(사→자), **し→じ(시→지)**
- **T→D**: た→**だ**(타→다), **ち→ぢ(치→지/디)**, **つ→づ(츠→즈/두)**
- **H→B/ P**: は→**ば/ぱ**(하→바/파)

# 6) 헷갈리는 쌍 요약(모양+소리)

- **し(시)** ↔ **つ(츠)** : し는 **왼쪽** 열림 / つ는 **오른쪽** 열림
- **そ(소)** ↔ **ん(응/n)** : **そ는 점이 붙고**, ん은 짧게 끝
- **ぬ(누)** ↔ **め(메)** : ぬ=고리+꼬리 / め=고리 단단히 닫힘
- **ね(네)** ↔ **れ(레)** : ね=막대+배(고리) / れ=고리 없음
- **わ(와)** ↔ **れ(레)** : わ 끝에 작은 점 느낌 / れ는 크게 휨

# 7) 아주 짧은 예문(읽기 연습)

- **きょう** → **쿄우**(오늘/오늘날; 今日)
- **しゃしん** → **샤신**(사진)
- **りょこう** → **료코-**(여행)
- **にゃん** → **냥/냥~**(고양이 울음 흉내)
- **おちゃ** → **오챠**(차)

조사(助詞, じょし)

## 1) は (와) : “은/는” (주제)

- **이 문장에서 지금 뭘 말하는지(주제)** 표시
- 발음은 “하”가 아니라 **“와”**

**예시**

- わたし**は**学生です。
(저**는** 학생이에요.)
- これはペンです。
(이것**은** 펜이에요.)

---

## 2) が (가) : “이/가” (주어, ‘바로 그게!’ 느낌)

- **누가/무엇이** 하는지
- 특히 “처음 등장하는 정보”, “강조”할 때 자주 써요

**예시**

- ねこ**が**います。
(고양이**가** 있어요.)
- だれ**が**来ますか？
(누**가** 와요?)

---

## 3) を (오) : “을/를” (목적어)

- **무엇을** 하다의 “을/를”
- 발음은 “오”

**예시**

- パン**を**食べます。
(빵**을** 먹어요.)
- 映画**を**見ます。
(영화**를** 봐요.)

---

## 4) の (노) : “의 / ~의 것 / 소유·소속”

- “A의 B”, “내 것” 느낌

**예시**

- わたし**の**本。
(내**의** 책 = 내 책)
- これはだれ**の**ですか？
(이건 누구**의** 거예요?)

---

## 5) に (니) : “에(시간/도착/대상)”

- **시간**: ~에
- **도착**: ~에(로) 가다/오다
- **대상**: ~에게(주다/말하다)

**예시**

- 3時**に**来ます。
(3시**에** 와요.)
- 学校**に**行きます。
(학교**에** 가요.)
- 先生**に**聞きます。
(선생님**께** 물어요.)

---

## 6) で (데) : “에서(장소) / 로(수단)”

- **행동이 일어나는 장소**: ~에서
- **수단/방법**: ~로, ~으로

**예시**

- 学校**で**勉強します。
(학교**에서** 공부해요.)
- バス**で**行きます。
(버스**로** 가요.)

---

## 7) へ (에) : “~로(방향)” (문어 느낌도 있음)

- 방향만 표시(도착은 확정 X)
- 발음은 “헤”가 아니라 보통 **“에”**처럼

**예시**

- 日本**へ**行きます。
(일본**으로** 가요.)

---

## 8) と (토) : “와/과”, 그리고 “~라고(인용)”

- **A와 B** 연결
- 말/생각의 **인용**

**예시**

- 友だち**と**行きます。
(친구**와** 가요.)
- 「行きたい」**と**言いました。
(“가고 싶다”**고** 말했어요.)

---

## 9) も (모) : “도”

- 추가: 나도/이것도

**예시**

- わたし**も**学生です。
(저**도** 학생이에요.)
- これ**も**おいしい。
(이것**도** 맛있어.)

---

## 10) から / まで : “부터 / 까지”

**예시**

- 9時**から**5時**まで**働きます。
(9시**부터** 5시**까지** 일해요.)

---

## 11) ね / よ : 문장 끝 “느낌”

- **ね**: “그쵸? 맞죠?” 공감/확인
- **よ**: “알려줄게/강하게 말함” 정보 전달

**예시**

- いい天気です**ね**。
(날씨 좋**네요**/좋죠?)
- これ、おいしいです**よ**。
(이거 맛있**어요(진짜)**.)

---

# 초보용 “바로 써먹는 세트”

- わたし**は** ○○です. (저는 ○○예요)
- ○○**が** あります/います. (○○가 있어요/있습니다)
- ○○**を** します/食べます/見ます. (○○를 해요/먹어요/봐요)
- ○○**に** 行きます / ○時**に** (○○에 가요 / ○시에)
- ○○**で** (○○에서 / ○○로)
